# pytest環境構築

## pytestインストール

```sh
pip install pytest
```

## ディレクトリ構成

```
my_project/
├── src/
│   ├── __init__.py
│   └── calculator.py       # テスト対象のソースコード
├── tests/
│   ├── __init__.py
│   ├── test_calculator.py  # テストファイル
│   └── conftest.py         # 共通 fixture（後述）
├── conftest.py             
├── pytest.ini              # pytest 設定ファイル

```

## pytest.ini設定

```sh
[pytest]
testpaths = tests       # テストディレクトリを指定
python_files = test_*.py
python_classes = Test*
python_functions = test_*
addopts = -v --tb=short  # 詳細表示・短いトレースバック
```

## conftest.py 

```py
import sys
sys.dont_write_bytecode = True
```

## テスト実行

```sh
# すべてのテストを実行
pytest

# 特定ファイルのみ
pytest tests/test_calculator.py

# 特定のテスト関数のみ
pytest tests/test_calculator.py::test_add_positive_numbers

# 詳細出力
pytest -v

# キーワードでフィルタ
pytest -k "add"
```

## assertの書き方
```py
# 等値比較
assert result == 42
assert result != 0

# 大小比較
assert result > 10
assert result <= 100

# True/False
assert is_valid == True
assert not is_error

# コレクション
assert 'key' in my_dict
assert item in my_list
assert len(my_list) == 3

# 型チェック
assert isinstance(result, int)

# 文字列
assert 'hello' in message
assert message.startswith('Hello')

# 近似値（浮動小数点）
import pytest
assert result == pytest.approx(0.1 + 0.2, abs=1e-6)
```
