## 論文概要：大規模言語モデルがソフトウェアエンジニアリングPBLに与える影響 - 2年間の縦断的実証研究

本研究は、ソフトウェアエンジニアリング（SE）プロジェクトベース学習（PBL）において、大規模言語モデル（LLM）の進化が学生のパフォーマンスに与える影響を評価した2年間の縦断的実証研究である。2024年度と2025年度の学生グループを比較し、ChatGPT 4oとGemini 2.5 Proという異なるLLMを導入した5週間のIPBL課題を実施した。

主な結果として、2025年度の学生グループはLLM支援下で、プロジェクト規模（CFP）が平均3.2倍に拡大し、相対分散も減少した。これは、LLMがより野心的で機能的なプロジェクトの開発を可能にしたことを示唆する。しかし、絶対的なパフォーマンスの分散（標準偏差）は拡大し、プロジェクトの規模に大きなばらつきが生じた。プログラミング能力（CFRP）はテスト合格率と中程度の正の相関関係を示し、基礎的なスキルが依然として重要である。

研究結果は、LLMが学生のプロジェクト規模を拡大する可能性を秘めている一方で、パフォーマンスのばらつきを増大させる可能性があることを示唆している。LLMを活用することで、プログラミングスキルに差がある学生でも、より複雑なソフトウェアエンジニアリングの課題に取り組む機会が提供され、PBLの質が向上する可能性がある。

しかし、LLMは高度なスキルを持つ学生の優位性をさらに拡大する「増幅器」として機能する可能性も示唆されており、教育格差を悪化させるリスクも存在する。教育設計においては、LLMの活用能力（戦略的な問題分解など）が新たな教育格差を生み出す可能性を考慮し、LLMとの協調的な実践に焦点を当てた教育設計が求められる。本研究は、SE教育においてLLMを効果的に統合するための課題と機会を浮き彫りにし、教育戦略の再考を促すものである。
