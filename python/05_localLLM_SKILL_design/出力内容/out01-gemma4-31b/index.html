The Concept

自然という、
最高の贅沢を。

私たちは、単なる宿泊場所ではなく、「時間を取り戻す場所」でありたいと考えています。 深い森の香りと、空の色が刻々と変わる風景。 デジタルな喧騒を離れ、五感がゆっくりと目覚める体験を。 ここでは、何もしないことが、最も贅沢な過ごし方です。
